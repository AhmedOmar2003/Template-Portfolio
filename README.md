# Universal Portfolio Template

A premium, bilingual portfolio template designed to work beautifully for any profession, including doctors, engineers, programmers, accountants, consultants, and creatives.

## Overview

This project is a modern responsive portfolio website built with plain HTML, CSS, and JavaScript. It focuses on a clean visual hierarchy, smooth animations, and a flexible layout that can be customized for different professional identities without rewriting the whole site.

## Key Features

- Bilingual interface: English and Arabic
- Responsive layout for desktop, tablet, and mobile
- Premium-looking hero section with strong visual impact
- Flexible sections for overview, strengths, capabilities, highlights, and contact
- Smooth scroll reveal animations
- Interactive testimonial slider
- Contact form validation
- Easy customization for colors, text, images, and branding

## Pages

- `index.html` - Main landing page
- `about.html` - About / profile section
- `services.html` - Capabilities and services
- `projects.html` - Work highlights and examples
- `contact.html` - Contact details and form

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome
- Google Fonts

## How To Run

You can open `index.html` directly in your browser, or serve the folder with a local live server extension for the best experience.

## How To Customize

1. Replace the placeholder name, logo, and contact details.
2. Update the hero text so it matches your profession.
3. Replace the example images with your own photos or brand visuals.
4. Edit the color palette in `style.css` if you want a different visual style.
5. Update the sample projects, testimonials, and timeline content.
